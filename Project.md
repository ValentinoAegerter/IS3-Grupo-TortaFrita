# Project: Sistema de Organización y Gestión de Eventos Académicos

## Objetivo del Producto
Desarrollar una aplicación web integral para que grupos de personas puedan organizar, gestionar y participar en eventos de tipo académico (cursos, jornadas, congresos, charlas, etc.). Se requiere una interfaz accesible desde la web para facilitar su uso desde cualquier dispositivo.

## Funcionalidades Principales (Alcance General)
* Gestión de eventos.
* Inscripción de participantes (autónoma y por el personal del evento).
* Gestión de roles (organizador, participante, disertante).
* Acreditación de participantes.
* Comentarios o encuestas de satisfacción post-evento.
* Generación de certificados (asistencia, aprobación, participación como autor/expositor).
* Generación de informes (incluyendo agenda del evento).
